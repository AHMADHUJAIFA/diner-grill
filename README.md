# Diner Grill 🍔

A sleek and modern HTML-based website for Diner Grill, showcasing a restaurant's menu, ambiance, and dining experience.

## 📋 Overview

Diner Grill is a static HTML website designed to provide an engaging online presence for a restaurant. The site features a complete, full-page design with a professional layout and user-friendly interface to attract customers and showcase the dining experience.

## 🎯 Features

- **Responsive Design** - Clean and professional layout optimized for various screen sizes
- **Complete Website** - Full-page HTML implementation with all essential sections
- **User-Friendly Interface** - Intuitive navigation and engaging visual design
- **Static Pages** - Fast-loading, lightweight HTML pages

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge, etc.)

### Installation & Usage

1. Clone the repository:
```bash
git clone https://github.com/AHMADHUJAIFA/diner-grill.git
cd diner-grill
```

2. Open the HTML file in your browser:
   - Simply double-click `840cc40b2_full_site.html` to open it in your default browser, or
   - Right-click and select "Open with" to choose a specific browser

3. Navigate through the website to explore the restaurant's offerings
